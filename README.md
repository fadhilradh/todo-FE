# todos.io Fullstack App

Stack : Next.js, Typescript, Tailwind, Redux Toolkit & Persist

Live : https://todos-io.vercel.app

Features :

- Register and login
- Non-login mode (data stored in localstorage)
- Post, delete and mark todo as done
